uso do iconic como imagem:

Referência no header
<link href="/open-iconic/font/css/open-iconic-bootstrap.css" rel="stylesheet">

<span class="oi oi-icon-name" title="icon name" aria-hidden="true"></span>

USO DOS ICONES NOVOS DO BOOTSTRAP

Opa, só avisando que agora é possível utilizar os ícones oficiais do Bootstrap 5, mas que podem ser utilizados também na nossa versão de estudos (4.1).

Basta que deixemos o link

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.4.0/font/bootstrap-icons.css">

para utilizarmos o cdn e implementemos um ícone svg como:

<i class="bi bi-calculator"></i>

Eles são muito bonitos, vale à pena conferir mais esta opção.
